## 기능 구현 실패 문제점

1. 안드로이드 스튜디오로 파이어베이스 데이터를 불러오기를 실패하였음.
2. UI 카테고리를 못 만듬(시간이 부족하였음)
3. 안드로이드 스튜디오 api level(29)과 sdk api level(28)이 달라서 애뮬레이터 실행 안됨.
   다운로드를 시도했으나 학교 서버가 너무 느려서 다운로드 실패함.
