<Skygear> - 오픈소스

Skygear Server는 웹, 모바일 및 IoT 앱 개발을보다 쉽게하기위한 클라우드 백엔드입니다.

일반적인 앱 개발을 위해 다음과 같은 기능을 제공합니다.

Skygear 인증-사용자 인증, 비밀번호 찾기 소셜 로그인 (출시 예정), 보안
Skygear Cloud DB-PostgreSQL 기반의 자동 마이그레이션 기능이있는 문서 형 API / SDK
Skygear 채팅-모든 기능을 갖춘 메시징을 앱 빠른 서버에 구축
Skygear Cloud 기능-서버에 대한 생각없이 이벤트 트리거 기능을 실행
Skygear CMS- 곧 Skygear Auth 및 Cloud DB Opensource 와 통합 된 비즈니스 사용자를위한 드롭 인 CMS
Skygear Pubsub-실시간으로 안정적인 메시징 및 연결을위한 Cloud Pub / sub API.
스카이 기어 푸시-푸시 알림
Skygear 서버 / 클라우드 기능은 클라이언트 측 SDK와 함께 사용됩니다.

자바 스크립트 : Core , Chat
iOS : 핵심 , 채팅
안드로이드 : 코어 , 채팅
Skygear.io 는 자체 인프라를 관리하지 않으려는 개발자를위한 상업용 호스팅 플랫폼입니다.

<apache usergrid> - 오픈소스

Usergrid는 웹 및 / 또는 모바일 애플리케이션을 빠르게 구축하려는 개발자를 위해 SDK가있는 통합 분산 NoSQL 데이터베이스, 애플리케이션 계층 및 클라이언트 계층으로 구성된 오픈 소스 백엔드 서비스 ( "BaaS"또는 "mBaaS")입니다. 기본 앱 (사용자 등록 및 관리, 데이터 저장, 파일 저장, 대기열) 및 검색 기능 (전체 텍스트 검색, 위치 정보 검색, 조인)을 제공하여 일반적인 앱 기능을 제공.

퍼블릭 클라우드 환경 (Amazon Web Services, Rackspace 등)에 배포하거나 기존 서버 인프라에서 실행하도록 설계되어 누구나 자신의 개인 BaaS 배포를 실행할 수 있도록 설계된 다중 테넌트 시스템

건축가 및 백엔드 팀을 위해 분산되고 쉽게 확장 가능하며 운영상 예측 가능하며 확장 성이 뛰어난 솔루션을 제공하는 것을 목표로합니다. 프론트 엔드 개발자의 경우, 백엔드 전문 지식 없이도 모바일 및 웹 애플리케이션을 신속하게 빌드하고 운영 할 수있게함으로써 개발 프로세스를 단순화하는 것을 목표로함

<Back4App> - 월/년 요금제별 요금 별도 (https://www.back4app.com/) 앱 + 데이터베이스 250 달러 ~ 
Back4App은 API, 데이터베이스, 파일 스토리지, 서버 인프라 등 모든 백엔드 구성 요소를 자동화하는 원 클릭 백엔드 플랫폼입니다. Back4App은 클라우드 기능, 푸시 알림, 사용자 관리, 백업 루틴 등과 같은 최신 응용 프로그램을위한 필수 백엔드 기능을 통합하여 개발자가 최소 80 % 시간 및 30 % 코드를 절약 할 수 있다.

Back4App 플랫폼은 가장 널리 사용되는 오픈 소스 백엔드 프레임 워크 인 Parse를 기반으로 합니다.
